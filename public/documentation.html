myFlix App - Documentation

This is my movie app where the user can find information about movies.

The table below describes each endpoint of this API and how to access it.

Description URL HTTP method Request body data format Response body data format
Get a list of all movies /movies GET None JSON object holding data about all movies
Get data about a single movie by title /movies/[title] GET None JSON object holding data about a single movie, containing title, description, genre, director,image URL
Get data about a movie genre /genre/[name] GET None JSON object holding data about a movie genre, containing name and description.
Get data about a director /director/[name] GET None JSON object holding data about a director, containing name, bio, birth year, death year.
Add a new user to the registry. /users POST JSON object holding data about the user to add, structured like:

{ username: "Johnny", password: "pw123", email: "johnny@gmail.com", dateofbirth: "20/12/92"

JSON object holding data about the user that was add, structured like:

{ id: 7, username: "Johnny", password: "pw123", email: "johnny@gmail.com", dateofbirth: "20/12/92"

Update a user's information /users/[username]/[password]/[email]/[date of birth] PUT None A text message indicating the user's information was updated.
Add a movie to the user's favorites list /favorites/[username]/[title] POST None A text message indicating the movie title was added to the user's favorites list.
Delete a movie from the user's favorites list /favorites/[username]/[title] DELETE None A text message indicating the movie title was deleted from the user's favorites list.
Remov a user from the registry. /users/[username]/[title] DELETE None A text message indicating the user was removed from the registry.
